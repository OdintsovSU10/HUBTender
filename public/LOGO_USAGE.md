# Логотип TenderHub by SU10

## Описание

Профессиональный логотип для системы управления строительными тендерами TenderHub by SU10.

## Концепция дизайна

Логотип объединяет несколько ключевых элементов:

1. **Строительная структура** - три здания разной высоты символизируют развитие и масштабность проектов
2. **Башенный кран** - символ строительства и прогресса
3. **Документ с чек-листом** - отражает тендерную документацию и управление процессами
4. **Бейдж SU10** - идентификатор компании на градиентном фоне
5. **Технологические линии** - намек на цифровизацию и современные технологии

## Доступные версии

### 1. Полный логотип (`/public/logo.svg`)
- **Размер**: 512x512 px
- **Использование**:
  - Favicon сайта
  - Главная страница
  - Презентационные материалы
  - Высокое разрешение для печати

### 2. Упрощенный логотип (`/public/logo-simple.svg`)
- **Размер**: 128x128 px
- **Использование**:
  - Иконки приложений
  - Аватары
  - Маленькие размеры (16px - 64px)
  - Мобильные интерфейсы

### 3. React-компонент (`/src/components/Icons/LogoIcon.tsx`)
- **Использование**:
  - Сайдбар навигации
  - Хедер приложения
  - Встраивание в React-компоненты
- **Возможности**:
  - Динамический размер через prop `size`
  - Поддержка светлой/темной темы
  - Настройка цвета (не рекомендуется менять)

## Цветовая схема

### Основные цвета

```css
/* Изумрудный зеленый (Primary) */
--primary-color: #10b981;

/* Темно-зеленый (Primary Dark) */
--primary-dark: #059669;

/* Циан (Accent) */
--accent-color: #0891b2;

/* Темный циан */
--accent-dark: #0e7490;

/* Белый для контраста */
--white: #ffffff;
```

### Градиенты

1. **Primary Gradient**: `#10b981 → #059669` (вертикальный)
2. **Accent Gradient**: `#0891b2 → #0e7490` (вертикальный)
3. **Crane Gradient**: `#059669 → #10b981` (горизонтальный)
4. **Badge Gradient**: `#10b981 → #059669 → #0891b2` (диагональный)
5. **Foundation Gradient**: `#059669(80%) → #10b981 → #059669(80%)` (горизонтальный)

## Правила использования

### ✅ Разрешено

- Использование в оригинальных цветах
- Масштабирование с сохранением пропорций
- Размещение на светлом и темном фоне
- Использование упрощенной версии для маленьких размеров

### ❌ Не рекомендуется

- Изменение цветовой схемы
- Искажение пропорций (растяжение/сжатие)
- Добавление эффектов (тени, обводки) без согласования
- Размещение на цветных фонах без проверки контраста
- Использование полной версии в размерах менее 32px

## Технические характеристики

### Формат файлов
- **SVG** - векторный формат, масштабируется без потери качества
- Поддержка градиентов и прозрачности
- Оптимизирован для веб-использования

### Размеры для разных контекстов

| Контекст | Рекомендуемый размер | Файл |
|----------|---------------------|------|
| Favicon | 16x16, 32x32, 48x48 | logo-simple.svg |
| Сайдбар | 24x24, 28x28 | LogoIcon компонент |
| Хедер | 32x32, 40x40 | LogoIcon компонент |
| Карточка | 64x64, 80x80 | logo-simple.svg |
| Страница | 128x128+ | logo.svg |
| Печать | 512x512+ | logo.svg |

## Примеры использования в коде

### React Component

```tsx
import { LogoIcon } from '@/components/Icons';

// В сайдбаре (маленький размер)
<LogoIcon size={28} color="#10b981" />

// В хедере (средний размер)
<LogoIcon size={40} color="#059669" />
```

### HTML

```html
<!-- Favicon -->
<link rel="icon" type="image/svg+xml" href="/logo.svg" />

<!-- Изображение -->
<img src="/logo.svg" alt="TenderHub by SU10" width="128" height="128" />
```

### CSS Background

```css
.logo-background {
  background-image: url('/logo-simple.svg');
  background-size: contain;
  background-repeat: no-repeat;
  background-position: center;
}
```

## Поддержка тем

Логотип хорошо работает в обеих темах:

- **Светлая тема**: зеленые оттенки контрастируют с белым фоном
- **Темная тема**: яркие градиенты выделяются на темном фоне

В компоненте LogoIcon цвет автоматически адаптируется:
- Светлая тема: `#059669` (темно-зеленый)
- Темная тема: `#10b981` (изумрудный)
